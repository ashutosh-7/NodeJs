# Running Locally
Make sure you have Node.js and the Heroku CLI installed.
```
  $ https://github.com/ashutosh-7/Online-Shop-Web-App.git # or clone your own fork
  $ cd E-coaching
  $ npm install
  $ npm start

```




# Deploying to Heroku 
``` 
  $ heroku login    
  $ heroku create
  $ git push heroku master
  $ heroku open
```
