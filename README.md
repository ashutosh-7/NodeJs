# Running Locally
Make sure you have Node.js and the Heroku CLI installed.
```
  $ https://github.com/ashutosh-7/E-coaching.git # or clone your own fork
  $ cd E-coaching
  $ npm install
  $ npm start

```




# Deploying to Heroku 
```
  $ sudo snap install --classic heroku   
  $ heroku login    
  $ heroku create
  $ git push heroku master
  $ heroku open
```
